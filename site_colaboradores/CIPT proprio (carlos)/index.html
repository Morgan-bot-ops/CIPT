Visão Geral

O projeto CIPT - Conecta: Informática para todos,consiste em, necessariamente, auxiliar pessoas que estão ingressando agora nas áreas pertinentes à Tecnologia da Informação(TI), com foco na criação de uma biblioteca virtual de repositório, tal que possa ensinar atémesmo quem não tem qualquer embasamento teórico na área.

Familiarizar os usuários com pautas relacionadas a:

  2. Web Development
  3. Web Design
  4. JAVA
  5. Segurança da Informação
